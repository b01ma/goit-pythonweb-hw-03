# goit-pythonweb-hw-03

## Project Overview

This project is a Python web application built with the standard library `http.server`.
It was created for a homework task that requires:

- Routing for two HTML pages: `index.html` and `message.html`
- Static file handling for `style.css` and `logo.png`
- Form processing on `message.html`
- Saving submitted messages to `storage/data.json`
- A `404 Not Found` fallback page (`error.html`)
- Running the application on port `3000`
- A `/read` route that renders saved messages using a Jinja2 template

## Implemented Features

- **Home page:** `GET /` and `GET /index.html` render `index.html`
- **Message page:** `GET /message` and `GET /message.html` render `message.html`
- **Form submit:** `POST /message` accepts `username` and `message`
- **Data persistence:** each form submission is appended to `storage/data.json`
- **Read page:** `GET /read` renders all saved messages via `templates/read.html` (Jinja2)
- **Static assets:** `GET /style.css` and `GET /logo.png`
- **404 handling:** unknown routes return `error.html` with HTTP status `404`

## Message Storage Format

Messages are stored in `storage/data.json` using a timestamp key generated by `datetime.now()`:

```json
{
	"2022-10-29 20:20:58.020261": {
		"username": "krabaton",
		"message": "First message"
	},
	"2022-10-29 20:21:11.812177": {
		"username": "Krabat",
		"message": "Second message"
	}
}
```

## Requirements

- Python `3.10+` (tested with Python `3.13`)
- `pip`

## Local Run

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python app.py
```

Open in browser:

- `http://localhost:3000/`
- `http://localhost:3000/message`
- `http://localhost:3000/read`

## Docker (Optional)

Build image:

```bash
docker build -t goit-pythonweb-hw-03 .
```

Run container with a mounted volume so `storage/data.json` is persisted outside the container:

```bash
mkdir -p storage
test -f storage/data.json || echo '{}' > storage/data.json
docker run --rm -p 3000:3000 -v "$(pwd)/storage:/app/storage" goit-pythonweb-hw-03
```

## Notes

- The app uses HTTP `303 See Other` after successful form submission and redirects to `/`.
- If `storage/data.json` is missing, it is created automatically as an empty JSON object.
